# NodeJS
Pequeña APP de consola con NodeJS
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base de la Tabla a Multiplicar.            [número] [requerido]
  -l, --list     Listar Resultados en Consola.       [booleano] [defecto: false]
  -e, --end      Limite, hasta que Numero Multiplicar.    [número] [defecto: 10]
```
